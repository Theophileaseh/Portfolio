## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites

``VS Code or other Code Editor``

``"Live server" configured``

### Install

``git clone https://github.com/Theophileaseh/Portfolio.git``

``cd Portfolio``

### Usage
